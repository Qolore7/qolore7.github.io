Graphics Base

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Graphics Base.
IMPORTANT: When installing graphics mods, the guide will generally suggest the highest-quality option that still retains good performance. If you have a weaker PC and/or want to squeeze maximum performance out of your rig, you can select the lower-quality options. However, it is NOT recommended to select higher quality options than what the guide recommends. These options usually have a minor visual upgrade compared to their performance cost and are not worth the lower performance and (potentially) stability.

NMCs Texture Pack

Installation instructions:

  1. Main File - NMCs Textures NV MEDIUM Pack Part 1 of 2 FOR NMM
  2. Main File - NMCs Textures NV MEDIUM Pack Part 2 of 2 FOR NMM
  3. Update Files - NMC New Vegas Patch for ALL Sized packs
  4. Update Files - Naval Chair Fix
  5. Once the files have been installed, double-click on the mod in the left pane of MO2 (specifically the Part 2 file if you did not merge them)
  6. In the Filetree tab, right-click on the Textures\interfance folder and select Delete
  7. Repeat for the Textures\landscape folder
Large-scale retextures of many major objects across the game, ranging from architecture to furniture.

Poco Bueno

Installation instructions:

  1. Main File - PocoBuenoTexturePackV5
  2. Optional File - PocoBueno Skeleton Transparency Fix
  3. Once the files have been installed, double-click on the mod in the left pane of MO2
  4. In the Filetree tab, right-click on textures\architecture\goodsprings\nv_watertank.dds and select Delete
  5. Repeat for textures\architecture\goodsprings\nv_watertank_n.dds
Large-scale retexture similar to NMCs.

6IXES Clutter Texture Pack

Installation instructions:

  • Main File - 6IXES Clutter Texture Pack NV Edition
Re-texture of many objects with a focus on smaller clutter items.

MGs Neat Clutter Retextures

Installation instructions:

  • Main File - MGs NCR Pack 7
  • Main File - BB Cigs
  • Main File - Common globe
High-quality of many objects overlooked by other texture packs.

Superpele's Disordered Texture Pack

Installation instructions:

  1. Download the file through Google Drive and install normally through MO2
  2. Once the files have been installed, double-click on the mod in the left pane of MO2
  3. In the Filetree tab, right-click on the textures\clutter\questitems folder and select Delete
Stylized retextures of a wide range of objects across the game.
