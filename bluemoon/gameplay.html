Gameplay

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.

Vigor - A Vanilla JSawyer Fork

Installation instructions:

  • Main File - Vigor
A fork of JSawyer, the mod made by the lead developer of the game, that focuses on a balance between the original mod's vision and vanilla gameplay and progression.

Real Recoil

Installation instructions:

  • Main File - Real Recoil v1.2.1
  • Main File - Real Recoil Re-Calculated
Adds realistic, dynamic, and smooth recoil to all ballistic firearms.

Unfound Loot

Installation instructions:

  • Main File - Unfound Loot
  • Main File - Unfound Loot INI
Dynamic and highly-customizable loot reduction.

Delay DLC Redux

Installation instructions:

  • Main File - Delay DLC Redux
Prevents DLC spam when starting a new game and requires you be near a given DLC's starting location to receive its quest.

Better Character Creation

Installation instructions:

  • Main File - Better Character Creation
Improves character creation by speeding up the process, adding specialized gear based on your tag skills, and making Wild Wasteland an opt-in feature rather than a trait.
