Miscellaneous Models & Textures

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Miscellaneous Models & Textures.

Prime Terminals - HD Terminal Replacer

Installation instructions:

  3. In the Filetree tab, delete Prime Terminals TTW - Vault 22 Terminal Fixes.esp
  4. Main File - Prime Terminals - YUP Patch
Replaces desk terminals with new HD models and textures.
