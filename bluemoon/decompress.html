Decompressing Records

Compression and Decompression

NPC, Navmesh, and Landscape records are automatically signed the Compressed flag to decrease their size footprint, but this can have an impact on performance and load times as decompressing is the most intensive part of loading. ESM Fixes already removes the compressed flag from the base game ESMs, but this process with remove them from all of your mods.

Installing FNVEdit

  1. Download Main File - FNVEdit 4.1.5 EXPERIMENTAL
  2. Once the download has finished, extract the contents of the archive to any location of your choice (do not use a UAC protected folder or the game folder).
  3. In MO2, select the button at the top of MO2 to open the executables menu.
  4. Click the + at the top of the window and the select Add from file...
  5. From the pop-up, navigate to where you installed FNVEdit and select FNVEdit.exe.
  6. In the Arguments box in the right pane, enter -IKnowWhatImDoing

Installing The Script

  1. Download Main File - Decompress Records
  2. Once the download has finished, extract the Edit Scripts folder from the archive to where you installed FNVEdit (there will already be an Edit Scripts folder in there that it should align with).

Running FNVEdit

  1. Select FNVEdit from the executables drop-down in the top right of MO2 and click Run.
  2. From the Module Selection pop-up, keep everything selected and hold down the Shift key and select OK.
  3. Select OK if there are any additional pop-ups and wait for the message Background Loader: finished to appear in the left pane.
  4. Press Ctrl+A to select everything in the left pane (you can optionally deselect the base game ESMs to save time).
  5. Right-click anywhere in the left pane and select Apply Script.
  6. From the script drop-down menu, select Decompress Records.pas then select OK.
  7. Wait for the message Done: Applying script to appear in the right pane (the script may take a while to complete based on how many plugins you have).
  8. Press Ctrl+S to save all edited plugins, then you can exit the program.
