lStewieAl's Tweaks

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator lStewieAl's Tweaks & Addons.

lStewieAl's Tweaks and Engine Fixes

Installation instructions:

  • Main File - Stewie Tweaks
Collection of many configurable quality of life improvements and bug fixes. Settings can be further enabled/disabled/tweaked in the new Tweaks section added to the main menu.

lStewieAl's Tweaks Preset

Installation instructions:

  • Main File - lStewieAl's Tweaks INI
Custom preset for lStewieAl's Tweaks made for this guide that enables many of the performance and quality-of-life improvements.

Auto-Save Manager

Installation instructions:

  • Main File - Save Manager
lStewieAl's Tweaks addon that fixes vanila auto-saves, adds timed autosaves, and save hotkeys

Engine Optimizations

Installation instructions:

  • Main File - Engine Optimizations
lStewieAl's Tweaks addon that rewrites various slow functions in the game to decrease loading/saving time and improve general performance.

Faster Main Menu

Installation instructions:

  • Main File - Faster Start Menu
lStewieAl's Tweaks addon that makes the games main menu load faster.

Faster Sleep Wait

Installation instructions:

  • Main File - Faster Sleep Wait
lStewieAl's Tweaks addon that decreases the delay between hours while waiting or sleeping.

Manual Reload and Ammo Memory

Installation instructions:

  • Main File - Manual Reload and Ammo Count Memory
lStewieAl's Tweaks addon that fixes ammo count being reset when switching weapons. Also optionally adds manual reloading, though the guide won't enable that option.

Menu Search

Installation instructions:

  • Main File - lStewieAl - Menu Search
lStewieAl's Tweaks addon that adds Ctrl-F to search in PipBoy, Container, Barter, Recipe, Levelup Perks and Save/Load menus.

Smooth True Iron Sights Camera

Installation instructions:

  • Main File - Smooth True Ironsights
lStewieAl's Tweaks addon that adds a smooth transition between idling and aiming down sights.

WASD Menu Controls

Installation instructions:

  • Main File - lStewieAl - WASD Menu Controls
lStewieAl's Tweaks addon that allows WASD for interaction in menus and adds TAB to go back or close most menus.

ActorCause Save Bloat Fix

Installation instructions:

  • Main File - ActorCause Save Bloat Fix
Fixes a save bloating issue related to actors detecting the player, improving save/load speed and decreasing save size 15-30% depending on playthrough length.

Better Caravan

Installation instructions:

  • Main File - Better Caravan
Significantly improves the usability of the Caravan minigame and fixes many bugs with the NPC AI.

Combat Lag Fix

Installation instructions:

  • Main File - Fast Weapon Lag Fix
Improves framerate in combat by fixing an engine bug that lowered framerate while attacking an enemy with a health-bar visible.

Console Paste Support

Installation instructions:

  • Main File - Console Paste
Adds Ctrl+V as a hotkey to paste into the in-game console, along with a few other helpful shortcuts.

Basic Console Autocomplete

Installation instructions:

  • Main File - Basic Console Autocomplete
Adds autocomplete and reverse searching to the game console.
