Weapon Models & Textures

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Weapon Models & Textures.

WAP Year One and Bonus

Installation instructions:

  • Main File - WAP Year One and Bonus - scripted
  • Main File - WAP Laser RCW - Main File
  • Main File - WAP Bozar and LMG - Main file
  • Main File - WAP Lever Action Shotgun
  • Main File - WAP F4NV Laser Pistol
  • Main File - WAP Sniper Rifle - Main file
  • Main File - WAP F4NV Recharger Weaponry - Main file
  • Main File - WAP Single Shotgun
Incredibly high quality model replacers for many weapons taken from ported from various sources.

ETJ Miniguns

Installation instructions:

  • Main File - No Camera ETJ Miniguns Vanilla Stats
New high-quality meshes, textures, sounds, and animations for the minigun and its unique variant.

Type II's Replacer Project

Installation instructions:

  • Optional File - Grenade Rifle Replacer
High-quality models and textures for the Grenade Rifle and all its variants based on their real-life counterpart.

Weapon Retxtures From EVE

Installation instructions:

  • Main File - Weapon Retexture from EVE
Standalone energy weapon retextures from Essential Visual Enhancements.

Weapon Enhancement Pack

Installation instructions:

  • Main File - Weapon Enhancement Pack
The mod is for Fallout 3, but works fine for New Vegas as well. If MO2 is having trouble using the Mod Manager Download, use the Manual Download option instead and install it like that.
In our case, this will mostly cover some energy weapons that aren't covered/aren't done as well by the other packs.

Fallout 3 Hirezd

Installation instructions:

  • Main File - Flamer
  • Main File - Minigun
Another overlooked Fallout 3 mod, but very high-quality stuff.

Simplified Weapon Retexture Project

Installation instructions:

  1. Optional File - Simplified WRP - WAP Cut
  2. Once the mods have been installed, double-click it> in the left pane of MO2
  3. In the Filetree tab, right-click on nvse folder and select Delete
Scratch-made retextures of a handful of weapons not covered by the other mods.

Melee Weapon Replacer Pack

Installation instructions:

  • Main File - Melee Weapon Replacer Pack
Sort of like WAP, but for melee weapons.

WJS's Super Sledge Ported

Installation instructions:

  1. Main File - Super Sledge v3
  2. Once the mod has been installed, double-click on it in the left pane of MO2
  3. In the Filetree tab, delete both files ending in .esp
  4. Main File - WJ's Super Sledge - Scriptrunner
New high-quality models for the Super Sledge and its unique variant ported from the F4NV project.

Fire Axe Replacer

Nail Gun Retexture
