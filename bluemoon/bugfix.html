Bug Fixes

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Yukichigai Unofficial Patch

Installation instructions:

  • Main File - YUP - Base Game and All DLC
Fixes thousands of bugs with both the base game and DLCs.

Unofficial Patch NVSE Plus

Installation instructions:

  • Main File - Unofficial Patch NVSE Plus
Fixes many bugs like YUP, but takes advantage of NVSE and JIP LN.

Landscape Texture Improvements

Installation instructions:

  1. Main File - Landscape Texture Improvements
  2. You will be prompted with a FOMOD installer:
    • FNV
      • Next
    • YUP Patch
      • Install
Fixes various texture seams in the landscape, most notably when using texture mods

New Vegas Mesh Improvement Mod

Installation instructions:

  • Main File - New Vegas Mesh Improvement Mod
Optimizations and fixes for over one thousand meshes throughout the base game and DLCs.

Misc Audio Tweaks and Fixes

Installation instructions:

  • Main File - Assorted Voice Popping Fixes
  • Main File - Elijah Missing Distortion Fix
A collection of tweaks for certain audio files to fix popping issues and other annoyances.

Strip Performance Fix

Installation instructions:

  • Main File - Strip Performance Fix
Drastically improves performance on The Strip by getting rid of water LOD.

Strip Street Light Corrections

Installation instructions:

  • Main File - Strip Street Light Corrections
Corrects several minor issues with the Strip's street lights.

Strip Lights Region Fix

Installation instructions:

  • Main File - Strip Lights Region Fix
Fixes a vanilla issue about the Strip lights not showing in certain parts of the map.

Lucky 38 Light Pollution Clip Fix

Installation instructions:

  • Main File - Lucky 38 Light Pollution Clip Fix
Fixes the misaligned light pollution emitted from the Lucky 38.

ExRB - Extended Roombounds

Installation instructions:

  1. Main File - ExRB
  2. You will be prompted with a FOMOD installer:
    • Full (YUP)
      • Next
      Install
Adds and tweaks roombounds in some interiors possibly improving performance.

Free Camp Golf Performance

Installation instructions:

  • Main File - free camp golf performance
Utilizes a number of under-used Gamebryo features to optimize performance around Camp Golf

Free Camp McCarran Performance

Installation instructions:

  • Main File - free camp mccarran performance
