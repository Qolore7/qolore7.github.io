Armor & Clothing Models & Textures

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Armor & Clothing Models & Textures.

4k Realistic Pip-Boy Retexture

Installation instructions:

  • Main File - Numb Squirrel Pip-Boy 4k Compressed
The mod is for Fallout 3, but works fine for New Vegas as well. If MO2 is having trouble using the Mod Manager Download, use the Manual Download option instead and install it like that.
Lore-friendly and realistic re-texture of the Pip-Boy 3000.

Power Armor Visual Enhancement

Installation instructions:

  1. Main File - Power Armor Visual Enhancement 2k
  2. Update File - PAVE Compatibility Plugins
  3. Once the mod has been installed, double-click on it in the left pane of MO2
  4. In the Filetree tab, delete PAVE_NV_TTW.esp
Custom high resolution textures for all the power armors in the game.

Wasteland Clothing HD

Installation instructions:

  • Wasteland Clothing HD V1-3 FULL PACK-54478-1-3.7z (Google Drive manual download)
High-quality up-scales with manual adjustments of many clothing and armors across the game.

Atompunk Vault Suits - HD Retexture

Installation instructions:

  • Main File - Atompunk Vault Suits
High-quality vanilla-plus re-texture of all the different types of vault suits.

ADAM Reborn

Installation instructions:

  1. Main File - ADAM All-In-One Installer 4.5.2
  2. Download the mod and install manually through MO2.
  3. You will be prompted with a FOMOD installer:
    • Custom Install
      • Next
    • NCR Trooper Armor
      • Next
      Next
      Install
  4. Once the mod has been installed, double-click on it in the left pane of MO2
  5. In the Filetree tab, delete all of the files ending in .esp
  6. Main File - ADAM Reborn - Plugin Replacer
Re-texture of almost all NCR-related armors in the game. The custom plugin from the guide disables any additional changes made outside of re-textures, such as new items.

Bornagain Combat Armor

Installation instructions:

  • Main File - Lite Version - BornagainFNVCombatArmor2048
  • Optional File - BornagainFNVCombatArmorLonesomeRoad4096
Re-texture of combat armor and its variants.

PM's HD Legion Overhaul

Installation instructions:

  • Main File - PM's HD Legion Overhaul - 2K
Beautiful hand-made re-textures of Legion armors that retain the vanilla aesthetics.

PM's HD Ranger Outfits

Installation instructions:

  • Main File - PM's HD Legion Overhaul - 2K
Beautiful hand-made re-textures of NCR Ranger clothing that retain the vanilla aesthetics.

F4NV Stealth Suits

Installation instructions:

  1. Main File - F4NV Stealth Suit
  2. Once the mod has been installed, double-click on it in the left pane of MO2
  3. In the Filetree tab, delete F4StealthSuitReplacerByUltra.esp
  4. Main File - F4NV Stealth Suits - Scriptrunner edition
Back-port of the Stealth Suit and Assassin Suit from the F4NV project.

Colorful Radition Suits

Installation instructions:

  • Main File - Red Radiation Suits
Up-scaled and re-colored radiation suits that look a bit more natural.
