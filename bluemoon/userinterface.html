User Interface

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator User Interface.

The Mod Configuration Menu

Installation instructions:

  • Main File - The Mod Configuration Menu
  • Update File - MCM BugFix 2
Allows select mods to be configured via a single in-game menu.

Vanilla UI Plus

Installation instructions:

  1. Download the mod and install manually through MO2.
  2. You will be prompted with a FOMOD installer:
    • Default Font Tweaks
    • WASD Compatible
      • Install
  3. Optional File - Vanilla UI Plus - Max Carry Weight Patch
Improves the user interface without compromising the original style.

JIP Improved Recipe Menu ESPless

Installation instructions:

  • Main File - JIP Improved Recipe Menu ESPless
Makes the crafting menu easier, more efficient and less tedious to use.

Clean Vanilla HUD

Installation instructions:

  1. Main File - Clean Vanilla Hud
  2. You will be prompted with a FOMOD installer:
      Install
  3. Main File - Clean Companion Wheel 256x256 Edition
  4. Main File - Clean Vault Boy Paper Doll
Hand-made up-scales of many HUD elements.

Just Dynamic Crosshair

Installation instructions:

  • Main File - Just Dynamic Crosshair
Adds a highly-customizable dynamic crosshair that reacts to spread.

Just Loot Menu

Installation instructions:

  • Main File - Just Loot Menu
Adds a highly-customizable quick-loot menu akin to Fallout 4/76.
The author of these two mods has a merged mod called Just Assorted Mods that contains the previous two mods along with many other features such as sprint and bullet time. The guide does not use this version as I don't personally like the rest of the additions, but feel free to use it if you want.

M.U.X. Series - Interface Overhaul

Installation instructions:

  1. Main File - M.U.X. Series - Interface Overhaul
  2. Optional File - M.U.X. Series - Just Dynamic Crosshair Patch
  3. Upon installing the main file, you will be prompted with a FOMOD installer:
    • Seventy Six
      • Install
  4. Main Files - OX - Toolkit
  5. You will be prompted with a FOMOD installer:
      Install
  6. Main File - M.U.X. INI
Overhauls the HUD to look more like the modern Fallout games.

Consistent Pip-Boy Icons

Installation instructions:

  • Main File - 1. Consistent Pip-boy Icons
Thousands of Pip-Boy icons re-made from scratch along with many bug fixes.

Clean Pause Background

Installation instructions:

  • Main File - Clean Pause Background
Simple HD retexture of the in-game pause menu

Cursor Unilaterally Matched NVSE

Installation instructions:

  • Main File - Cursor Unilaterally Matched (NVSE) - Vanilla
Makes the game's cursor match your HUD color

Simple Cursor (YUI)

Installation instructions:

  • Main File - Simple Cursor (SMALL)
A simple and clean cursor replacer for CUM. You can choose the (LARGE) version if you prefer.

Main Title Consistent Replacer

Installation instructions:

  • Main File - Main Title Replacer - New Vegas
Simple HD replacer of the game's main title.

Minimal Atmospheric Loadscreens

Installation instructions:

  • Main File - Minimal Atmospheric Loadscreens
Simple and sleak AI-generated load screen images. Use the Low Res version if you prefer.

Simple Main Menu HD

Installation instructions:

  • Main File - Simple Main Menu HD - 2K
Replaces the main menu with a new high-definition image. You can use whichever size variant you prefer.

World Map HD

Installation instructions:

  • Main File - World Map HD
Crisp HD up-scale of the games world maps.

ESP-less Tutorial Killer

Installation instructions:

  • Main File - ESP-less Tutorial Killer
Disables tutorial pop-ups and adds a resource to disable/tweak messages from any mod via an INI file.
