Animations

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Animations.

ISControl Enabler and Ironsights Adjuster

Installation instructions:

  • Main File - ISControl
Dynamically adds ISControl nodes to all weapons which is required for most animation mods and adds configuration of iron sight alignment per-weapon.

HIPControl - Weapon Idle Position Adjuster

Installation instructions:

  • Main File - HIPControl
Allows you to easily configure weapon idle/hipfire position per-weapon.

NV Compatibility Skeleton

Installation instructions:

  • Optional File - NVCS
A skeleton replacer with automatic compatibility with many mods.

Faster Pip-Boy Animation

Installation instructions:

  • Main File - Faster Pip-Boy Animation (2x)
Doubles the speed of the Pip-Boy opening animation.

B42 Weapon Inertia

Installation instructions:

  • Main File - B42 Inertia V1.2
Adds smooth weapon inertia physics.

DiaMove NVSE Patched

Installation instructions:

  • Main File - DiaMoveNVSE Patched
NVSE Plugin that adds simple diagonal movement animations for the player.

Anniversary Anim Pack

Installation instructions:

  • Main File - Anniversary Anim Pack
Simple and smooth animation replacers for most weapon-related animations.

Anniversary Anim Pack - General Bugfix

Installation instructions:

  • Main File - AnniAnimPack_BugFix 1.3
  • Main File - Bonus Patch
Bug fixes for Anniversary Anim Pack.

FNV Clean Animations - Reborn

Installation instructions:

  • Main File - FNV Clean Animations - Part 1 (R)
  • Main File - FNV Clean Animations - Part 2 (no mesh changes)
  • Main File - 3. No TTW & Update File - Update 1.1.2 - HOTFIX
    1. Once the mods have been installed, double-click on FNV Clean Animations - Part 3 in the left pane of MO2
    2. In the Filetree tab, right-click on DoubleBarrelShotgun.esp and select Hide
Utilitizes kNVSE to give many weapon beautiful unique animations.

Butcher Pete Complete - A Melee Animation Overhaul

Installation instructions:

  • Main File - 2. Butcher Pete Complete - A Melee Animation Overhaul (No Locomotion)
Utilitizes kNVSE to give almost every one-handed melee weapon beautiful unique animations.

Wasteland Warrior - A Melee Animation Overhaul

Installation instructions:

  • Main File - 2. Wasteland Warrior - A Melee Animation Overhaul (No Locomotion)
  • Update File - Update 1.0.1
Utilitizes kNVSE to give almost every two-handed melee weapon beautiful unique animations.

Blended Locomotion

Installation instructions:

  • Main File - Blended Locomotion
Blends multiple locomotion (movement animation) sets into one clean package.

Animated Vault 21 Hotel Sign

Installation instructions:

  • Main File - Vault22AnimatedSign
Adds a spinning animation to the vault door on the Vault 21 Hotel sign.

Freeside Structure Rotation Animation

Installation instructions:

  • Main File - Freeside Structure Rotation Animation Main
Adds rotation and glowing lights to the large Freeside sign.

Lucky 38 Casino Chips Go Spin

Installation instructions:

Adds a spinning animation to the casino chips on th Lucky 38 Casino sign.
