Creature Models & Textures

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Creature Models & Textures.

10 Year Anniversary Celebration Pack

Installation instructions:

  1. Main File - Main File
  2. Update File - FriedTurkey's Mantis Add On
  3. Once the mod has been installed, double-click on it in the left pane of MO2
  4. In the Filetree tab, delete 10YearPack.esp
  5. Repeat for meshes\creatures\nvmantis\nvmantis.nif
  6. In the meshes\creatures\nvmantis folder, rename the nvmantis_FT_less_shiny.nif file to nvmantis.nif
Updates many models and textures for vanilla creatures using assets from Fallout 4 mods and other great artists. The plugin is deleted as to only keep the new creature models, as the armor models lack female variants and have potential stability issues.

TheFriedTurkey 3 Creature Pack

Installation instructions:

  • Main File - Main Archive
  • Main File - Mojave Bighorners
Replaces various creatures with high-quality models from the F4NV project.

Securitrons On Alert

Installation instructions:

  • Main File - Securitrons On Alert
Adds new animated faces for Securitrons when they enter combat.

Securitrons Lip Sync

Installation instructions:

  1. Main File - Securitrons Lip Sync
  2. In the Filetree tab, delete NV_Securitrons_Lip_Sync.esp
  3. Repeat for the textures folder
Adds lip sync to Securitrons when they're talking.
